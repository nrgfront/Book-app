# 📚 Book Finder App (React + Vite)

A simple and modern book search application built with **React, Vite, and Open Library API**.  
Users can search books, view details, and save favorites using localStorage.

---

## 🚀 Features

- 🔍 Search books using Open Library API
- 📄 View detailed book information
- ❤️ Add/remove favorites
- 💾 Save favorites in localStorage
- 📱 Responsive UI
- ⚡ Fast performance with Vite

---

## 🛠️ Tech Stack

- React
- Vite
- React Router
- CSS Modules
- Open Library API

---

## 🌐 Deployment
